Do-Hickey is a tool that employs the use of advanced machine learning algorithms, within image preprocessing, character recognition, language modelling, post-processing and page layout analysis to successfully extract text from images

To do this it uses tesseract-ocr a tool for this intended purpose. Only that it trained once more and fine tuned to adjust to this data and collect accuracte information.

The process of collecting this text from a document is highlighted below.

  • Upon the successful upload of document to Do-Hickey, background task is triggered that starts the processing of the file for character extraction.
  • The first task that is triggered involves the conversion of the document into high resolution image(s)
  • After a successful conversion, the image(s) go throuogh an upscaling process to help make small detail visible
  • From here, the image(s) has its dpi increared and then finally fed into the OCR algorigthm
  • The OCR algorithm extracts the text and then saves it to the database.

And that's it, the document would have been processed and parsed completely. The best part about all of this is that it is very well optimized whihc means very fast loading and processing times.

All of the steps it takes to extarct data here usually takes anywhere between 5 seconds to 5 minutes in total and is usually dependent on the document size.
